# DuckSeason_bHaptics

This repository was forked from Astienth/DuckSeason_bHaptics to add owoskin compatibility to the game Duck Season.


# Description

This mod adds owoskin support to the game

# Installation

1. **Download** [MelonLoader.Installer](https://github.com/HerpDerpinstine/MelonLoader/releases/latest/download/MelonLoader.Installer.exe) and **install melon loader**.

2. Download this mod from the releases section.

3. Unpack the zip-file and run the containing executable.

4. Select your Duck Season game executable and click on install mod

5. Connect your owoskin

6. Start the game and feel the pain
